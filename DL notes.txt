## ReLU (Rectified Linear Unit)
Advantages: Fast computation (no exponentials), sparse activation, and avoids vanishing gradients for positive inputs.
Disadvantages: Can suffer from "Dying ReLU" (neurons become inactive permanently) if learning rates are too high or data is negative.
Best Use: Hidden layers of deep convolutional and feed-forward networks.



## Back Propagation :

Loss function, Cost function

This process only works on DL ==> Supervised learning 

Loss function == it calculate the only one data loss (actual value - predicated)

Cost function == it will calculate the over all data in dataset and give the average value 

Gradient Descent:

an iterative optimization algorithm used to minimize the cost function (error) in machine learning models by updating parameters (weights and biases) in the direction of the steepest decrease. It works by calculating the gradient of the loss function, scaled by a learning rate, to gradually find the minimum, typically used to train linear regression and neural networks

Global minima are the lowest absolute points of a function over its entire domain, while local minima are lower than surrounding points but not necessarily the lowest overall. A function can have only one global minimum, but multiple local minima. Optimization algorithms often aim for global minima, but can get stuck in local minima

Epoch weight are carry forward to next epoch ?

The weights learned after each epoch are carried forward to the next epoch, allowing the model to progressively improve rather than starting from scratch.

Epoch Early stopping ?

Early stopping halts training when validation performance stops improving, preventing overfitting and saving computation time.
